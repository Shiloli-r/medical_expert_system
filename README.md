## How to Run
<ol>
<li> Open the Project Folder </li>
<li> Click the CLIPSIDE64.EXE and open the application </li>
<li> Go to `File` > `Load` > `Project folder` > `expert_system.clp` </li>
<li> Go to `Execution` > `Run` </li>
<li> Interact with the program</li>
</ol>
